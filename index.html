Propiedades Básicas de CSS en Acción

Este proyecto demuestra cómo aplicar diferentes estilos para mejorar la apariencia de una página web.

Ejemplos de Estilos

Colores y Fondos

Uso de `background-color` para el fondo y `color` para el texto. ¡Mira qué bien se ve!

Márgenes y Rellenos

La propiedad `margin` crea espacio alrededor de un elemento, mientras que `padding` crea espacio adentro.

Bordes y Sombras

Con `border-radius` redondeamos las esquinas y con `box-shadow` añadimos una sombra sutil para dar profundidad.
